CNAT Design System

House Style · v1.0 · July 2026

One system. Every CNAT page.

Distilled from the AI Builds Field Manual and the class landing pages. Load one stylesheet, pick a theme, and everything you build looks like us.

How to use on any page
<link rel="stylesheet" href="https://classes.cnattitle.com/brand/cnat.css">
<html data-cnat-theme="bright">   <!-- or manual-light / manual-dark -->
<body class="cn">                 <!-- add .cn-longform for reading pages -->

Palette

Tokens change with the theme; class names never do. Brass/gold is the only decorative accent and is spent sparingly. Rust means warning or a human gate, never decoration.

Type

Fraunces for display and italic structural numerals. Libre Franklin for UI and uppercase letterspaced labels. Charter/Georgia for long-form body. Monospace for money and data.

Kicker label above a headline

Display with an italic accent.

The lede paragraph is italic Fraunces, muted, and never wider than 62 characters.

Long-form body text is set in Charter or Georgia at 17px with generous leading. It is for guides and editorial pages, not UI chrome.

House Rules

RuleMeaning
Square cornersNo border-radius anywhere. No pill shapes, ever.
Rules over boxesHairlines and double rules structure the page. Shadows only on Bright cards.
One accentBrass/gold, spent sparingly. Rust only for warnings and human gates.
Typography is the brandFraunces display + italic numerals, Franklin labels, serif long-form, mono data.
No em dashesCommas, periods, or "to". Applies to every word on every page.
44px touch targetsMobile-first, always. Chip navs wrap instead of scrolling.
Humans on the buttonsRust marks every human gate in flows. Nothing auto-sends.
